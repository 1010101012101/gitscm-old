Download

To obtain a copy of Git:

  • Source tarballs and RPMs are available at http://www.kernel.org/pub/software/scm/git/.
  • Daily snapshots are available at http://www.codemonkey.org.uk/projects/git-snapshots/git/ (thanks to Dave Jones).
  • If you already have Git installed, you can get the latest development version via Git itself: 
    git clone git://git.kernel.org/pub/scm/git/git.git
  • If you have problems connecting, you can try to get it via Git itself but over the HTTP protocol: 
    git clone http://git.kernel.org/pub/scm/git/git.git
    (this method is considerably slower but works even behind firewalls and such).
