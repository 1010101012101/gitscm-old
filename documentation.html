Documentation

You can find all the documentation bundled with the Git source code in the Documentation/ subdirectory. In addition, you can also read the web version (corresponding to the latest Git development version).

For a quick start, the tutorial covers the basics. See the Everyday Git handbook for a useful minimum set of commands depending on your style of Git usage. Additionally, the Kernel Hacker's Git Tutorial by Jeff Garzik may be useful.

More as an experiment right now, we have recently set up a Git Wiki - you are welcome to contribute!
